# Phase 1 MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the Phase 1 Jiandanly MVP baseline: authenticated chat, streaming model gateway, local-first client history, billing status, Stripe subscription entry, deployment scaffolding, tests, and documentation.

**Architecture:** The backend is a Go API gateway with storage and provider interfaces so local development can run on deterministic in-memory/mock adapters while production can connect PostgreSQL, DeepSeek/OpenAI-compatible providers, Anthropic Claude, and Stripe. The frontend is a shared React/Vite app with web and Electron shells, IndexedDB local history, API/SSE clients, and export/import tools.

**Tech Stack:** Go `net/http`, JWT, bcrypt, PostgreSQL migrations, React, TypeScript, Vite, Vitest, Electron, IndexedDB, Docker Compose.

---
